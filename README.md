## 기능

- 글 작성, 요약, 오타수정, 코딩 도우미, 번역 등의 AI 기능
- 텍스트, 이미지 저장
- 스티키 노트
- 테이블
- 코드 스니펫 저장
- 갤러리
- 나의 노트


## 시작하기


1. pnpm install 패키지 설치
2. env 환경 변수 설정
3. pnpm dev -> localhost:3000
